## Install Guide
```bash
$ git clone https://github.com/qulc/bootcamp.git

$ cd bootcamp/

$ pip install -U -r requirements.txt

# Create Table
$ python manage.py makemigrations
$ python manage.py migrate

# Run
$ python manage.py runserver
```
